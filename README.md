:::info
# 專案簡介
人在海外，想找個社運組織取暖，卻遍尋不著嗎？讓自己成為同溫層的中心吧！這個資料庫專案結合各種海外社運組織列表、曾經參與的行動、進行流程圖表、讀書會歷年書單和電子檔、哲五歷年講者和題目、各種線上/TECO紀錄片取得管道、以及各種組織疑難雜症解方，讓搞社運就像在subway點餐一樣容易！
